Point Operations

  • If we multiply the values by a constant > 1, contrast of the image increases, and the width of the histogram increases (the range)
  • If we multiply by a constant < 1, the contrast will decrease and the width of the histogram will decrease.
  • Adding or subtracting constant values from the intensity values of the image, the histogram will shift. Note that any negative values are removed, so the histogram is cut off/terminated.
  • Shades of grey can either be
    • Limited (black and white, or black and white and a mid value)
    • A continuous gradient
  • Both will have the same contrast, as contrast represents the difference between the lowest value (black) and the highest value (white).
  • Dynamic range is the number of unique values that are present in the image. So if any midpoint value between the highest and lowest values is not present, it is not counted with the dynamic range. So in this case the dynamic range will differ in the above two cases.

How can we enhance images?

We can utilize a method called histogram equalization.

It should only be used when the image is over or underexposed, not otherwise.

  1. Map the old intensity levels to the new intensity level values to get the new image histogram. For example, if we learn from the equation that intensity levels A and B map into intensity level C, then the new pdf for C will include the values that were there for A and B. If no level maps into intensity level D, then its new value will be 0.
